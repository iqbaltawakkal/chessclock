# Feature

- Accurate timestamp.
- Various time types (bullet, rapid, blitz, classical & custom).
- Flip timer.
- Night Mode.
- Progressive Web App (PWA) Support!

# Stack

- Framework - Next.js.
- Styling - Tailwind CSS + React Icons (Material icon).
- Component Lib - HeadlessUI Tailwind.
